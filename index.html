Cómo publicar esta app
Gratis en GitHub Pages · 10 minutos
Esta app funciona como un único archivo HTML. Súbela a GitHub Pages y tendrás un enlace permanente que cualquier persona puede abrir desde el celular o computador, sin instalar nada.
1
Crea una cuenta en GitHub
Ve a github.com y regístrate gratis. Solo necesitas correo y contraseña.
2
Crea un repositorio nuevo
Haz clic en el botón verde "New". Ponle el nombre exacto:
combustibles
Marca la casilla "Add a README file" y haz clic en "Create repository".
3
Sube el archivo app_combustibles.html
Dentro del repositorio, haz clic en "Add file → Upload files". Arrastra el archivo app_combustibles.html y luego haz clic en "Commit changes".
4
Renombra el archivo a index.html
Haz clic en el archivo subido, luego en el ícono del lápiz (editar), cambia el nombre a index.html en la parte superior y guarda con "Commit changes".
5
Activa GitHub Pages
Ve a Settings → Pages (menú lateral). En "Source" selecciona Deploy from a branch, elige la rama main y carpeta / (root). Haz clic en Save.
6
Espera 2 minutos y copia el enlace
GitHub te mostrará el enlace. Será algo como:
https://tu-usuario.github.io/combustibles/
Comparte ese enlace con los 8 usuarios. Funciona en cualquier celular o computador, sin instalar nada.
⚠️
Limitación importante del Camino A
Cada usuario guarda sus registros solo en su propio dispositivo (memoria del navegador). Los datos no se sincronizan entre personas automáticamente. Para consolidar, cada uno debe usar el botón Exportar CSV y enviarte el archivo para que lo pegues en el Excel central.
Si necesitas que todos vean los mismos datos en tiempo real, el Camino B (base de datos compartida) resuelve eso.
